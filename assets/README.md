# Portfolio images

Place these files in this folder (same names as in `index.html`):

| File | Used for |
|------|----------|
| `logo.png` | Sidebar logo |
| `profile.jpg` | About section photo |
| `onlinemarketing.jpg` | NEARBY MARKET project |
| `smartpoultry.jpg` | SMART POULTRY project |
| `travelblog.jpg` | TRAVEL GUIDES project |
| `coding.jpg` | Achievements (if referenced) |
| `hackathon.jpg` | Achievements (if referenced) |

If a project image is missing, the browser shows a placeholder message.

**Deploying:** Commit this `assets/` folder to GitHub so images work on GitHub Pages / Netlify. Files only on your PC will not appear on the live site.
